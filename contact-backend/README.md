# Contact Backend

Simple Node + Express + MongoDB backend for a contact page.

Quick start:
1. copy .env.example to .env and set MONGO_URI and PORT.
2. npm install
3. npm run dev
4. POST to /api/contact with JSON { name, email, message }
# from repo root
npm install
cp .env.example .env
# edit .env to set MONGO_URI if needed, then:
npm run dev
